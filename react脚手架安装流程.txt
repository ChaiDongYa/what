
	这节课的学习内容  react脚手架  

	第一步 ->
		1. 首先打开我们的指令器 cmd ->
		2. 安装 yarn ->
			-> 输入 cnpm i yarn -g
			-> 下载完了后 不知道行不行 能不能用 那好 咱们检查一下
			-> yarn -v 检查 yarn 是否存在(已下载)
		3. 然后安装 react-app
			-> 输入 cnpm i create-react-app -g
		
	安装好了后->
		1. cd 一个文件夹 下载脚手架到文件夹
		2. 输入 create-react-app 自定义项目名称 
			-> 下载脚手架
	
	下载好了脚手架到本地文件夹之后->
		1. 重新 cd 一下 刚才自定义的 脚手架文件夹
		2. 在输入 yarn start  这时候 就会自动运行网页 
			-> 运行的地址就是 react脚手架给配置好的地址:
				-> http://localhost:3000
		3. 哈哈...完事啦~~ 
	
	下载React-脚手架的版本还有其他方式 -->
